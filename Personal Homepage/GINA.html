INFORMATION

GINA (Generations of Nuclear Activity) is a program created at the National Superconducting Cyclotron Laboratory (NSCL). It's purpose is to perform calculations of nuclear decay rates for a neutron rich isotope chain in order to determine the contamination that is produced over time.

FEATURES

GINA is programmed in Qt and designed solely to function on Linux macines.

Current features included in GINA include but are not limited to the following.

  • Determines numerical values for particle numbers, decay rates, and contamination for a nuclear isotope beam deposited onto a target.
  • Plots data over time in a labeled and fancy manner and allows the user to save the plots.
  • Allows the user to save analysis as a data file with the delimiter "|".
