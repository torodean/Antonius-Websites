INFORMATION

The best way to learn coding is by coding.

MIA (Multiple Integrated Applications) is a program where I can store useful coding functions I've created in one place for easy accessibility. It is a project I've started to better improve my programming skills and give me something to work on in my free time.

MIA is still in the early beta programming stages.

FEATURES

MIA is a cross platform terminal program that runs on both Windows and Linux. MIA contains various random features. MIA is very new and a lot of already coded (as part of other programs) features are still in the process of being integrated with MIA.

Current features included in MIA include but are not limited to the following.

  • String Encryption through a custom d0s1/d0s2 algorithm with upcoming file encryption (d0s3) in development.
  • Various mathematical functions such as add/subtract/multiply functions for integers of any length.
  • Prime number generation and ordered prime number list generation.
  • Windows Key code simulation with numerous Minecraft features for performing tasks while afk.
  • More and more to come...

For a complete list of features you can view the MIA manual at the following link.
